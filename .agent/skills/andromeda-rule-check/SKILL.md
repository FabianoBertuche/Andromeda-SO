# Objetivo
Fazer checagem de conformidade.

# Instruções
1. Liste as rules aplicáveis à tarefa.
2. Verifique a proposta contra cada rule.
3. Marque status: ok, risco, bloqueio.
4. Sugira ajustes se houver conflitos.

# Saída
- rules aplicáveis
- conformidade
- bloqueios
- ajustes necessários
