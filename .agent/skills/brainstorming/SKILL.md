# Objetivo
Clarificar o problema antes de convergir para uma implementação.

# Instruções
1. Liste objetivo, restrições e hipóteses.
2. Gere poucas opções viáveis, não uma chuva de confete arquitetural.
3. Compare trade-offs.
4. Recomende a opção mais simples que resolva o problema.
