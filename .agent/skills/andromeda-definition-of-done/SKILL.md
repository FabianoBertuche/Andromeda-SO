# Checklist
- o escopo pedido foi atendido?
- regras aplicáveis foram respeitadas?
- testes foram criados/atualizados?
- regressões prováveis foram consideradas?
- documentação mínima foi atualizada?
- riscos remanescentes foram declarados?

# Instrução
Nunca encerrar sem passar por este checklist.
