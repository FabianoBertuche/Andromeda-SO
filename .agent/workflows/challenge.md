# Challenge

## Use quando
- a solução parece complexa demais
- há mais de uma abordagem plausível
- a mudança toca áreas sensíveis

## Passos
1. Liste a proposta atual.
2. Ataque a proposta:
   - é simples o bastante?
   - quebra padrões atuais?
   - gera acoplamento?
   - cria risco de segurança?
   - realmente precisa dessa complexidade?
3. Compare alternativas.
4. Escolha a abordagem mais simples que resolva o problema.
