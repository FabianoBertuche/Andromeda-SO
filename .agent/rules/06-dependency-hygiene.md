# Dependency Hygiene

## Objetivo
Evitar dependências desnecessárias, frágeis ou arriscadas.

## Regras
- Não adicione pacote sem necessidade real.
- Prefira stdlib ou solução interna para casos triviais.
- Justifique toda dependência nova.
- Verifique impacto, manutenção e segurança antes de introduzir pacote.
